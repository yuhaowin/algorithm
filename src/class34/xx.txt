题目一
32位无符号整数的范围是0-4294967295,现在有一个正好包含40亿个无符号整数的文件， 可以使用最多1GB的内存，怎么找到出现次数最多的数?

1GB = 10亿byte 字节
假设 1GB 内存可以支持 1千万的 size 的 hash map
最大有 42亿种数 / 1千万 = 400
把这个文件中的数 hash 后 % 400，写入 400 个文件
通过 hash map 统计出每一个文件中出现最多的数
